Client:
To compile the client, a recent version of qt(along with qt-creator dependancies) is needed.
The Makefile located in source/client/source/build can be used to compile the client into the 'client' executable.

Server:
To compile the server, run the makefile in source/server/c 
Now run the executable

Note:
The encryption algorithm is available in both ascii and obfuscated ascii forms in the files
enc.c and enc_deobfs.c respectively.
